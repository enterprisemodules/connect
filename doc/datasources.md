# Available datasources

The Connect language contains a number of datasources that are distributed as separate puppet modules:
- [Yaml](https://github.com/hajee/connect_yaml/blob/master/README.md)
- [Encrypted](https://github.com/hajee/connect_encrypted/blob/master/README.md)
- [Puppedb](https://github.com/hajee/connect_puppetdb/blob/master/README.md)
- [DNS](https://github.com/hajee/connect_dns/blob/master/README.md)
- [Consul](https://github.com/hajee/connect_consul/blob/master/README.md)
- [LDAP](https://github.com/hajee/connect_ldap/blob/master/README.md)
